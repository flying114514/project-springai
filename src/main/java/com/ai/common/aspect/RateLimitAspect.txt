用户请求 → Controller 方法
    ↓
┌─────────────────────────────────────────────┐
│ @Around 拦截 (@RateLimit 注解)               │
├─────────────────────────────────────────────┤
│ 1. 解析注解参数                              │
│    - dimensions: [GLOBAL, IP]               │
│    - count: 5                               │
│    - interval: 1 (秒)                       │
│                                            │
│ 2. 计算时间窗口                             │
│    intervalMs = 1000ms                     │
│                                            │
│ 3. 生成 Redis Keys                          │
│    - ratelimit:{...}:global                │
│    - ratelimit:{...}:ip:192.168.1.100      │
│                                            │
│ 4. 调用 Lua 脚本                             │
│    KEYS: [key1, key2]                      │
│    ARGS: [timestamp, 1, window, max, uuid] │
│                                            │
│ 5. Lua 脚本内部逻辑                          │
│    ┌──────────────────────────────────┐    │
│    │ for each key:                    │    │
│    │   ZREMRANGEBYSCORE (清理过期)    │    │
│    │   ZCARD (统计当前窗口请求数)      │    │
│    │   if count > max: return 0       │    │
│    │   ZADD (添加新请求)              │    │
│    │ return 1                         │    │
│    └──────────────────────────────────┘    │
│                                            │
│ 6. 判断结果                                │
│    result = 1 ✅ → 执行原方法               │
│    result = 0 ❌ → 执行降级方法或抛异常     │
└─────────────────────────────────────────────┘
